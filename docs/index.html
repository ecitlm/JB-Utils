web-utils

常用前端js工具库

Installation

Using npm:

$ npm i --save @ecitlm/web-utils

how to use

// 全部引入
import * as webUtils from '@ecitlm/web-utils';
webUtils.getUID();

//按需引入
import { getUID } from '@ecitlm/web-utils';
